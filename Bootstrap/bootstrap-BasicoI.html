Reboot (reinicialização)

A reinicialização baiseia-se apenas na padronização de estilo para alguns elementos do HTML, fazendo o uso de seletores CSS pelo nome da tag e atributos.

Página padrão e fontes nativas

Os elementos <html> e <body> são modificados para melhor exibição da página.

Parágrafos

O elemento HTML <p> (ou elemnto de parágrafo) representa um parágrafo do texto.


Listas

O elemento HTMl <ul> (ou elemento de lista desordenada) representa uma lista de items sem ordem rígida, isto é, uma coleção de items que não trazem uma ordenação numérica e as suas posições.

  • Risole
    • Risole de Queijo
    • Risole de presunto com queijo
  • Pastel
    • Pastel de Queijo
    • Pastel de pizza
  • Lasanha
    • Lasanha a bolonhesa
    • Lasanha 4 queijos

O elemento HTML <ol> (ou elemento de lista ordenada) representa uma lista de items ordenados.

  1. Strognoff de cogumelo
  2. Batata empanada
  3. Pizza

O lemento HTML <dl> (ou elemento de lista de definições) engloba uma lista de pares de termos e descrições.

Frutas
Banana, abacaxi e uva
Legumes
Berinjela, chuchu, batata

Textos pré-formatados

O elemento HTML <pre> representa o texto pré-formatado que deve ser apesentado exatamente como está escrito um arquivo HTML

        * texto pré-formatado *
